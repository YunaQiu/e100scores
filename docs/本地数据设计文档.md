# localStorage
* bankAlias:题库题目,json格式：
```
array=>[									//题目列表
	object=>{
		title: (str) 						//标题
		type: (int) 						//题型编号
		options: array=>[					//选项列表
			object=>{
				option: (str) 				//选项内容
				correct: (tinyint) 			//是否正确
			}
		]
		key: (str) 							//正确答案
		analysis: (str) 					//答案解析
	}
]
```

* bankAlias-user：题库的所有用户答案,json格式：
```
//version 1.0
array=>[					//题目列表
	array=>[				//选项列表
		(tinyint) 			//是否选择
	]
]

//version 2.0
object=>{
	version: (str) 			//版本号
	record: array=>[		//答题列表
		object=>{
			checked: array=>[
				(tinyint) 	//是否选择
			]
		}
	]
	updateTime: (str) 		//更新日期
}
```
